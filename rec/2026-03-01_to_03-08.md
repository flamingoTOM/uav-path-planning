### 3月6日 — Hybrid A* 算法完整实现

**14. Hybrid A* 路径规划算法**

本周最重要的算法产出。基于自行车运动学模型，在连续 (x, y, θ) 状态空间中进行启发式搜索，生成平滑曲线路径，更符合实际无人机运动约束。

核心设计：

| 参数 | 值 |
|------|---|
| 状态空间 | 连续 (x, y, θ) |
| 运动模型 | 自行车运动学 |
| 步长 `step_size` | 3.0 |
| 转向角离散数 `n_steers` | 5 |
| 最大转向角 `max_steer` | 0.6 rad |
| 轴距 `wheel_base` | 3.5 |

- 新增文件：`example/python_motion_planning/path_planner/graph_search/hybrid_a_star.py`（223 行）
- 同步更新安装包路径，确保 runtime 可用
- 生成效果图：`pic/hybrid_astar.png`
