# 随机碰撞覆盖法子技能

## 功能

实现和可视化随机碰撞覆盖算法（Random Bounce Coverage）。

## 算法原理

机器人在封闭区域内：
1. 沿固定方向直行
2. 遇到障碍物或边界时随机转向
3. 继续直行
4. 重复上述过程实现区域覆盖

## 特点

- **优点：** 对定位与建图要求低，计算开销小
- **缺点：** 产生重复轨迹，整体效率较低
- **应用：** 扫地机器人、区域巡逻等

## 使用方法

```bash
# 基本用法（10次碰撞）
python pyscripts/random_bounce_algorithm.py --bounces 10

# 自定义参数
python pyscripts/random_bounce_algorithm.py \
    --bounces 20 \
    --width 150 \
    --height 80 \
    --output custom_bounce.png
```

## 可视化内容

- **封闭边界** - 黑色实线框
- **障碍物** - 灰色填充区域
- **运动轨迹** - 蓝色连续路径
- **起点** - 绿色圆点
- **终点** - 红色方块
- **碰撞点** - 红色圆点，带序号标注

## 参数说明

- `--bounces, -n`: 碰撞次数（默认：10）
- `--width`: 区域宽度（默认：100）
- `--height`: 区域高度（默认：60）
- `--output, -o`: 输出文件路径
- `--show`: 显示图形窗口

## Python API

```python
from random_bounce_algorithm import RandomBounceVisualizer

viz = RandomBounceVisualizer(width=100, height=60)
fig, path, bounces = viz.visualize(
    start_pos=(5, 5),
    num_bounces=10,
    save_path='pic/bounce.png'
)
```
