# 性能基准测试子技能

## 功能

对算法进行多次运行，收集统计数据，评估性能。

## 测试指标

- **执行时间** - 平均值、标准差、最小值、最大值
- **路径成本** - 平均值、标准差
- **路径长度** - 平均节点数
- **扩展节点数** - 搜索效率指标

## 使用方法

```bash
# 默认测试（10次迭代）
python subskills/performance_benchmark.py

# 指定算法和迭代次数
python subskills/performance_benchmark.py --algorithms a_star dijkstra --iterations 20

# 自定义输出文件
python subskills/performance_benchmark.py --save results.txt
```

## 输出格式

### 控制台输出
```

dijkstra        87.46±6.12    137.882±0.00      119         3421
```

### 文件输出
保存到 `pic/benchmark_results.txt`，包含详细统计信息。

## 最佳实践

1. **采样算法** - RRT、RRT* 是随机算法，建议增加迭代次数（50+）
2. **图搜索算法** - 确定性算法，10次迭代足够
3. **对比分析** - 在相同地图上测试多个算法

## Python API

```python
from performance_benchmark import benchmark_algorithm

result = benchmark_algorithm('a_star', iterations=20)
print(f"平均时间: {result['time_mean']:.2f}ms")
print(f"平均成本: {result['cost_mean']:.3f}")
```
