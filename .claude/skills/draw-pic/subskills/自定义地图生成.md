# 自定义地图生成子技能

## 功能

创建不同模式的自定义地图用于算法测试。

## 支持的地图类型

### 1. 迷宫地图
**特点：** 垂直和水平障碍物形成迷宫结构
**适用场景：** 测试算法在复杂环境中的寻路能力

### 2. 窄通道地图
**特点：** 多个窄通道形成瓶颈
**适用场景：** 测试算法通过狭窄空间的能力

### 3. 螺旋地图
**特点：** 螺旋形障碍物模式
**适用场景：** 测试算法绕行能力

## 使用方法

```bash
python subskills/custom_map_example.py
```

## 输出

为每种地图类型和每种算法生成独立的可视化图片：
- `maze_<算法>_<时间戳>.png`
- `narrow_passage_<算法>_<时间戳>.png`
- `spiral_<算法>_<时间戳>.png`

## Python API

```python
from custom_map_example import create_maze_map, create_narrow_passage_map

# 创建迷宫地图
map_obj = create_maze_map()

# 创建窄通道地图
map_obj = create_narrow_passage_map()
```

## 自定义地图示例

```python
builder = MapBuilder(width=120, height=80)

# 添加自定义障碍物
builder.add_obstacle_line(x=50, y=(10, 70))
builder.add_obstacle_rectangle(x=30, y=30, width=20, height=10)

map_obj = builder.build(start=(10, 10), goal=(110, 70))
```
